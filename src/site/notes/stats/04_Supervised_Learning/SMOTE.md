## Definition

> [!abstract] Overview
> **SMOTE (Synthetic Minority Over-sampling Technique)** is a technique used to address imbalanced datasets. Instead of simply duplicating minority class examples (oversampling), SMOTE generates *synthetic* examples by interpolating between existing minority instances.
