## Definition

> [!abstract] Overview
> **Generalized Linear Models (GLM)** extend ordinary linear regression to allow the response variable to have an error distribution other than a normal distribution. GLMs fit data by maximizing the likelihood of the observed data.

## key Components
1. **Random Component:** Probability distribution of the response variable (e.g., Normal, Binomial, Poisson).
2. **Systematic Component:** Linear predictor ($X\beta$).
3. **Link Function:** Connects the random and systematic components.
